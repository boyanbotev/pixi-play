# Flappy Plane

This repository contains the code and assets for 'Flappy Plane,' a game developed to demonstrate the use of PixiJS, Spine2D and GSAP.
### [Demo Build](https://boyanbotev.itch.io/flappy-plane)

### Features
- Flappy Bird gameplay with increasing speed
- GSAP motiontween animation when plane hits a cloud
- Spine2D animations for the clouds and plane
- Endless background scrolling

### Technologies
- [Webpack](https://webpack.js.org/) as a bundler
- [PixiJS](https://pixijs.com/) as the rendering engine
- [GSAP](https://greensock.com/gsap/) for animation
- [TypeScript](https://typescriptlang.org/) for type safety
- [Spine2D](https://esotericsoftware.com) for animation
